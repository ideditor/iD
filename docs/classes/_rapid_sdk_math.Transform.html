Class Transform

Transform is a class for dealing with transform data x,y - translation, (from origin coordinate [0,0], to top-left screen coordinate) k - scale, (related to the map zoom, how many Mercator coordinates the world contains) r - rotation, optionally applied post-projection to change the map bearing away from north-up
